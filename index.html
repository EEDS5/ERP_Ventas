Introducción

ERP Ventas es un sistema de gestión de ventas y facturación electrónica. Facilita la administración de clientes, ventas, cuentas por cobrar y reportes empresariales.

El proyecto sigue una arquitectura robusta basada en principios Clean Architecture y SOLID, dividida en backend y frontend que se comunican mediante API REST.

Características principales

  • Backend: Spring Boot 3.4.4, PostgreSQL, JasperReports, JWT y 2FA.
  • Frontend: Angular 19 con Angular Material.
  • Arquitectura hexagonal en ambos módulos.
  • Contenerización completa con Docker y Docker Compose.
  • Documentación de API generada con Swagger/OpenAPI.
